# jquery报错：Unexpected end of input
`【系统环境】debian`  
`【操作内容】jquery`  
`【解决方案验证次数】1次`  
## <i class="fa fa-question-circle"></i> 出现问题
写jquery时候出现错误：
```
Unexpected end of input
```
## <i class="fa fa-bullseye"></i> 根本原因
括号或者大括号没对齐。
## <i class="fa fa-check-circle"></i> 解决方法
找到缺失的括号或者大括号，让括号对应，错误就消失了。
