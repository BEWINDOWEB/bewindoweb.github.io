## Hello World
```lua
print("hello world!")
```

## 脚本式编程
```lua
#!/usr/local/bin/lua
print("hello world!")
```

## 常用语法

### 注释
单行注释：
```lua
-- 我是注释
```
多行注释：
```lua
--[[
<html>
   <div>hello lua</div>
</html>
--]]
```
