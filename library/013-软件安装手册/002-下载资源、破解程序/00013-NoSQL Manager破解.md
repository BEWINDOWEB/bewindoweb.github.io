# NoSQL Manager 破解

## 破解方法1
当试用期满的时候：  
（1）删除注册表项
```
HKEY_CURRENT_USER\Software\NoSQL Manager Group
```
（2）删除应用数据
```
C:\ProgramData\NoSQL Manager Group
```
（3）如果还不行，删除
```
C:\Users\Administrator\AppData\Roaming\NoSQL Manager Group\NoSQL Manager for XXX（MongoDB、Cassandra、……）
```
